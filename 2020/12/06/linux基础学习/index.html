linux基础学习

linux文件系统

linux是类unix系统，一个很重要的概念就是万物皆文件，包括我们执行的shell命令等都是存储在文件当中；

win中习惯用斜杠,而linux习惯用反斜杠/;

/bin，目录下存放的是二进制可执行文件，比如 ls 就是在 /bin/ls 目录下的

/boot，存放的是启动 Linux 时使用的一些核心文件，包括一些连接文件以及镜像文件

/etc ，用来存放所有的系统管理所需要的配置文件和子目录

/home ，用户主目录，几个用户就有几个文件夹

/var，var 是 variable(变量) 的缩写，这个目录中存放着在不断扩充着的东西，我们习惯将那些经常被修改的目录放在这个目录下。包括各种日志文件。

/usr， usr 是 unix shared resources(共享资源) 的缩写，这是一个非常重要的目录，用户的很多应用程序和文件都放在这个目录下，类似于 windows 下的 program files 目录。

/opt，opt 是 optional(可选) 的缩写，这是给主机额外安装软件所摆放的目录

/tmp，tmp 是 temporary(临时) 的缩写这个目录是用来存放一些临时文件的。

重定向

linux下系统打开3个文件，即标准输入、标准输出和标准错误输出。用户的shell将键盘设为默认的标准输入，将屏幕设为默认的标准错误输出。

重定向就是不使用系统默认的标准输入，输出，而是重新指定；

  • 输入重定向，操作符为“<”和“<<”;将文件中内容代替键盘输入；

    “<<”操作符作用？？？

  • 输出重定向，“>”,“>>”,将本应输出的屏幕的内容输出到指定文件；

  • 错误重定向，操作符“2>”，“2>>”,将错误信息输出到指定文件，而不是屏幕；

管道

管道可以将很多命令连接起来，作用是：将一个命令的输出当作下一个命令的输入；管道符： |

引用

为了将特殊字符当作普通字符处理，linux使用了引用技术来忽略这些字符的特殊含义，shell中用于引用的字符有斜杠\、单引号‘、双引号“这三个；
